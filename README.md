# 🌦️ Weather App

A simple and interactive Weather App built using **JavaScript** that allows users to check the **current weather conditions of any city** in real-time.

---

## 🚀 Features

* 🔍 Search weather by city name
* 🌡️ Displays temperature, humidity, and weather condition
* 🌥️ Shows dynamic weather status (Clear, Clouds, Rain, etc.)
* ⚡ Fast and responsive UI
* 📱 User-friendly interface

---

## 🛠️ Tech Stack

* **HTML** – Structure
* **CSS** – Styling
* **JavaScript** – Functionality
* **Weather API** – Fetch real-time weather data

## ⚙️ How It Works

1. User enters a city name
2. App sends request to weather API
3. API returns weather data
4. Data is displayed on the screen

## 🧑‍💻 Installation & Setup

1. Clone the repository:

```bash
git clone https://github.com/your-username/weather-app.git
```

2. Navigate to the project folder:

```bash
cd weather-app
```

3. Open `index.html` in your browser

---

## 🔑 API Setup

* Get a free API key from a weather service (like OpenWeatherMap)
* Replace your API key in the JavaScript file:

```js
const API_KEY = "your_api_key_here";
```
